# Filled Proof Story Submissions

Place real project notes in this folder.

Recommended process:

1. Copy `../proof-story-intake-template.md`.
2. Paste the copy into this folder.
3. Rename it using a safe internal label.
4. Fill out whatever you know.
5. Mark the publication confidence clearly.

Example filenames:

- `industrial-parts-response-agent.md`
- `private-childcare-front-desk-voice.md`
- `home-services-missed-call-recovery.md`
- `professional-services-intake-routing.md`
- `marketing-content-approval-workflow.md`

Do not place confidential screenshots, client names, credentials, private records, or unredacted transcripts in this folder unless the repository access model is appropriate for that material.
