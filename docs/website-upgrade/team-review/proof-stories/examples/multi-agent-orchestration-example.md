## Before State

A customer request might start as a call, continue by email, require a CRM update, need a calendar booking, trigger an internal task, and require staff approval.

Each step depended on a person remembering to move information from one system to the next.

## What The Automation Did

- Voice agent captured the initial request.
- Classification agent identified intent, urgency, and missing details.
- CRM agent updated or created the record.
- Calendar/task agent prepared booking or follow-up.
- Email/SMS agent sent approved next-step messaging.
- Operations agent routed exceptions.
- Human reviewer approved sensitive actions.
- Event log captured the workflow outcome.

## Human Controls

Human-owned:

- Approval thresholds.
- Customer commitments.
- Pricing.
- Sensitive decisions.
- Escalation policy.
- Final workflow expansion decisions.

The agents should coordinate around approved rules rather than independently improvising commitments.

Safe outcome language:

- Fewer dropped handoffs between channels.

Metrics to collect:
